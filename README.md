# RFM Customer Segmentation Project

## Objective
To segment customers using RFM Analysis (Recency, Frequency, Monetary).

## Steps Performed
- Data Cleaning
- Removed returns
- Created Total Amount
- Calculated RFM Metrics
- Assigned Scores
- Segmented Customers
- Visualized Results

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Insights
- Identified top customers
- Detected at-risk customers
- Suggested marketing strategies
